# Personal website

Personal website for me :)

Website template from `alshedivat/al-folio`.


## Run locally

To run and edit the web locally run:

```bash
git clone https://github.com/ianchu0317/ianchu0317-web.git
cd ianchu0317-web
docker compose up -d
```

And preview in `http://0.0.0.0:8080/al-folio/`
